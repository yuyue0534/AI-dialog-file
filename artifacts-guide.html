基本概念

Artifacts 是什么？

过去，你和 Claude 对话，回答只是一段文字。而 Artifacts 改变了这一切——它让 Claude 能够直接在对话旁边，生成一个可以预览、运行、下载的独立作品。

简单来说：Artifacts 就是 Claude 帮你"做出来"的东西，不只是"说给你听"的答案。

它可以是一个网页、一张图表、一段可运行的代码、一份精美文档……你可以直接看到它、用它、改它。

想象你请一位助手帮你写一份合同。以前他会把文字念给你听；现在，他直接把打印好的合同递到你手上。这就是 Artifacts 的意义。

使用方法

怎么用 Artifacts？

使用 Artifacts 不需要任何技术背景，只需用自然语言描述你想要什么：

  1. 打开 Claude，正常对话
    进入 claude.ai，在对话框里用中文（或任何语言）描述你的需求。不需要特殊命令。
  2. 描述你想要"做出来"的东西
    关键是让 Claude 知道你需要一个可交付的成果，而不只是解释。例如："帮我做一个…" / "创建一个…" / "生成一份…"
  3. Artifact 出现在右侧面板
    Claude 生成内容时，Artifact 会在对话右侧实时显示。网页会直接运行，文档会直接渲染。
  4. 预览、下载、或继续修改
    满意就下载使用；不满意就直接告诉 Claude "把颜色改成蓝色" / "加上标题" ——它会直接在 Artifact 上修改。
💡

新手小技巧： 不确定怎么说？直接描述你的目标场景就好，比如"我需要向老板汇报销售数据，帮我做一个直观的图表"，Claude 会自动判断最合适的 Artifact 类型。

注意事项

开始前，了解这几点

✅

Artifacts 默认开启——在 claude.ai 的设置中，确认"Artifacts"功能已打开（通常默认开启）。

🔄

可以反复修改——生成后不满意？直接在对话框继续说"把字体改大"、"加一个深色模式"，Claude 会更新同一个 Artifact。

📱

手机也能用——Claude 移动端 App 同样支持 Artifacts，生成的内容可以直接分享或保存。

⚡

AI 驱动——高级 Artifacts（如 AI 对话应用）可以内嵌 Claude API，让你的 Artifact 本身就具备 AI 能力。这是进阶玩法，适合有兴趣的用户深入探索。
